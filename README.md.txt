# Acortador de URLs con Flask

Este es un proyecto sencillo hecho con Python y Flask que permite convertir una URL larga en un enlace corto.

## Descripción

La aplicación recibe una URL ingresada por el usuario, genera un código corto aleatorio y permite redirigir al enlace original usando ese código.

Este proyecto fue creado como práctica para aprender Flask, rutas, formularios HTML y redirecciones.

## Tecnologías usadas

- Python
- Flask
- HTML
- CSS
